# mgonzalezimp-web

Sitio **de una sola página** (scroll) para publicar en Netlify.

## Estructura
- index.html
- styles.css
- script.js
- assets/
  - hero.jpg (placeholder)
  - trabajo-01.jpg ... trabajo-06.jpg (opcionales para galería)

## Antes de publicar
1. Si deseas, sustituye `assets/hero.jpg` por una imagen real (logo/obra).
2. Si quieres mostrar la sección "Trabajos", agrega imágenes con estos nombres:
   - assets/trabajo-01.jpg
   - assets/trabajo-02.jpg
   - ...
   - assets/trabajo-06.jpg

## Publicación en Netlify (rápido)
- Arrastra y suelta la carpeta en Netlify (Deploy manually), o conecta un repositorio Git.
